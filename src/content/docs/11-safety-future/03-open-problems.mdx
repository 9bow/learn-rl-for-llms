## 서론: 해결되지 않은 핵심 질문들

LLM을 위한 강화학습은 빠르게 발전하고 있지만, 근본적인 질문들은 여전히 열려 있습니다. 이 장에서는 현재 연구 커뮤니티가 직면한 7가지 핵심 열린 문제를 검토하고, 각각의 유망한 연구 방향을 탐구합니다.

## 열린 문제 1: RLVR의 추론 경계

**질문**: RLVR(Reinforcement Learning with Verifiable Rewards)은 진정한 추론 능력을 확장하는가, 아니면 단지 기존 지식을 더 효율적으로 압축/활용하는가?

### 확장 주장 (Expansion)

RLVR가 훈련 데이터에 없던 새로운 추론 패턴을 발견한다는 주장:
- DeepSeek-R1은 RLVR 훈련 중 자발적으로 "자기 반성(self-reflection)" 패턴을 습득했습니다.
- 훈련 시 보지 못한 문제 유형에서도 성능 향상이 관찰됩니다.
- 추론 토큰 수가 증가할수록(더 많이 생각할수록) 더 어려운 문제를 해결합니다.

### 압축 주장 (Compression)

RLVR가 사전 훈련에서 획득한 지식을 더 효율적으로 활용하는 것뿐이라는 주장:
- 사전 훈련 데이터에 없는 완전히 새로운 개념은 RLVR로도 학습되지 않습니다.
- 성능 향상이 관찰된 벤치마크들은 사전 훈련 코퍼스와 높은 분포 유사성을 보입니다.
- 작은 모델에서 RLVR의 효과가 크게 줄어드는 것은 기존 능력의 활성화로 해석됩니다.

### 현재 이해

두 주장 모두 부분적으로 맞을 가능성이 높습니다. RLVR는 기존 지식을 더 효율적으로 활용하면서(압축), 동시에 새로운 조합과 추론 패턴을 발견하는(확장) 복합적인 과정으로 보입니다. 이를 명확히 분리하는 실험적 프로토콜의 설계가 필요합니다.

## 열린 문제 2: 보상 일반화

**질문**: 검증 불가 도메인에서 어떻게 신뢰할 수 있는 보상 신호를 설계하는가?

RLVR는 수학, 코드처럼 정답을 자동으로 검증할 수 있는 도메인에서 강력합니다. 그러나 창의적 글쓰기, 오픈엔드 질의응답, 전략적 의사결정 등은 검증 자체가 어렵습니다.

### 현재 접근법과 한계

| 접근법 | 설명 | 한계 |
|--------|------|------|
| **보상 모델** | 인간 피드백으로 훈련된 RM | 보상 해킹, 분포 이탈 |
| **헌법적 AI** | 원칙 기반 자기 평가 | 원칙 자체의 모호성 |
| **LLM-as-Judge** | 강한 LLM이 평가 | 편향, 자기 선호 |
| **앙상블** | 여러 평가자의 합의 | 동일한 편향 공유 가능성 |

### 유망한 방향

**과정 보상(Process Reward)**: 최종 결과뿐 아니라 추론 과정의 각 단계를 평가. 검증 불가 도메인에서도 논리적 일관성, 사실성, 관련성 등을 단계별로 평가할 수 있습니다.

**역강화학습(Inverse RL, arXiv:2507.13158)**: 전문가의 행동에서 보상 함수를 역산하는 접근법. 검증 불가 도메인에서 전문가 데이터로부터 암묵적인 평가 기준을 추출할 수 있습니다.

## 열린 문제 3: 멀티모달 정렬의 고유 도전

**질문**: 이미지, 비디오, 오디오 등 다양한 모달리티를 포함하는 RLHF는 어떻게 달라야 하는가?

텍스트 전용 RLHF와 멀티모달의 핵심 차이:

- **보상의 모달리티 불일치**: 이미지를 생성했을 때의 보상을 텍스트 기반 보상 모델로 평가하면 정보 손실이 발생합니다.
- **시각적 환각(Visual Hallucination)**: 모델이 이미지에 존재하지 않는 객체를 "보는" 현상. RLHF로 완화 가능하지만 완전한 해결은 어렵습니다.
- **시간적 일관성**: 비디오 이해에서 프레임 간 일관성을 보상에 반영하는 방법이 불명확합니다.

## 열린 문제 4: 에이전트 안전 — 자율 행동의 통제

**질문**: 도구를 사용하고 장기적 목표를 추구하는 자율 에이전트를 어떻게 안전하게 제어하는가?

에이전트 RL의 고유한 위험:

- **되돌리기 불가능한 행동**: 파일 삭제, 이메일 발송, API 호출 등은 취소가 불가능합니다.
- **목표 미스얼라인먼트**: 에이전트가 "주어진 목표를 달성하라"는 지시를 의도와 다르게 해석할 수 있습니다.
- **인간 승인 회피**: 작업 완료를 위해 사람의 확인이 필요한 단계를 우회하는 방법을 발견할 수 있습니다.

### 접근법

**행동 제약(Action Constraints)**: 에이전트가 수행할 수 있는 행동의 유형과 범위를 명시적으로 제한합니다. 불가역적 행동은 인간 승인을 요구합니다.

**최소 권한 원칙**: 현재 태스크에 필요한 최소한의 권한만 에이전트에게 부여합니다.

**RLHF for Agents**: 에이전트의 장기 행동 시퀀스에 대해 인간이 피드백을 제공하는 방식으로 정렬합니다.

## 열린 문제 5: 대규모 학습 효율성

**질문**: 수백 B 파라미터 모델의 RLHF를 실용적 비용으로 수행하는 방법은?

현재 상태:
- GPT-4 수준 모델의 전체 RLHF 훈련은 수백만 달러의 컴퓨팅 비용이 필요합니다.
- 대부분의 연구팀은 7B~70B 규모 모델에만 접근 가능합니다.

유망한 방향:
- **LoRA 기반 RLHF**: 전체 파라미터의 1% 이하만 업데이트
- **오프라인 RLHF**: 온라인 상호작용 없이 기존 데이터로 정렬
- **지식 증류를 통한 정렬 전이**: 정렬된 큰 모델의 선호를 작은 모델로 전이

## 열린 문제 6: 인간 감독의 확장

**질문**: AI 능력이 증가함에 따라 인간 감독자의 수와 전문성이 함께 증가해야 하는가?

현재 RLHF 파이프라인에서 인간 피드백은 병목입니다:
- 고품질 주석자는 희귀하고 비용이 많이 듭니다.
- 도메인 전문 지식이 필요한 태스크(의학, 법률, 수학)는 전문가 주석이 필수입니다.
- AI 능력이 주석자를 초과하면 피드백의 품질이 의미 없어집니다.

이 문제는 앞서 다룬 Scalable Oversight와 직결됩니다. AI를 활용한 피드백 생성(RLAIF), 과정 보상 모델, Debate 기반 평가 등이 인간 감독의 확장을 지원할 수 있습니다.

## 열린 문제 7: 다목적 최적화

**질문**: 유용성(Helpfulness), 안전성(Safety), 정직성(Honesty) 등 여러 목표를 동시에 최적화하는 방법은?

이 세 목표는 종종 충돌합니다:
- 안전성을 높이면 유용성이 감소합니다 (Alignment Tax)
- 정직성을 높이면 때로는 사용자가 원하지 않는 진실을 말해야 합니다
- 유용성을 극대화하면 잠재적으로 해로운 정보도 제공할 수 있습니다

### 현재 접근법

**선형 보상 조합**:
$$r_{total} = w_1 r_{helpful} + w_2 r_{safe} + w_3 r_{honest}$$

가중치 $w_1, w_2, w_3$의 설정이 핵심이며, 이는 사회적 가치 판단을 내포합니다.

**Pareto 최적화**: 단일 솔루션이 아닌 Pareto 최적 프론티어를 찾아 사용 맥락에 따라 선택합니다.

**계층적 목표**: 안전성을 절대 제약(hard constraint)으로, 유용성을 최적화 목표(objective)로 설정합니다.

## 미래 연구 방향

| 열린 문제 | 유망한 접근법 | 시간 지평 |
|-----------|--------------|----------|
| RLVR 추론 경계 | 반사실적 실험 설계, 능력 추적 | 단기 (1-2년) |
| 보상 일반화 | 역강화학습, 과정 보상 모델 | 중기 (2-3년) |
| 멀티모달 정렬 | 멀티모달 보상 모델, 크로스모달 RLHF | 중기 (2-3년) |
| 에이전트 안전 | 행동 제약 + 형식 검증 조합 | 장기 (3-5년) |
| 학습 효율성 | LoRA-RLHF, 지식 증류 | 단기 (1-2년) |
| 인간 감독 확장 | Scalable Oversight + AI 보조 주석 | 중기 (2-3년) |
| 다목적 최적화 | Pareto RLHF, 계층적 제약 | 중기 (2-3년) |

### Inverse RL for LLMs (arXiv:2507.13158)

가장 유망한 새로운 방향 중 하나는 **역강화학습(Inverse Reinforcement Learning, IRL)** 의 LLM 적용입니다. 전통적 IRL은 전문가의 행동 시퀀스를 관찰하여 암묵적인 보상 함수를 추론합니다.

LLM 맥락에서 IRL은 다음을 가능하게 합니다:
- 인간이 명시적으로 정의하기 어려운 선호를 전문가 텍스트에서 역산
- 다양한 문화와 맥락에서의 선호 학습
- 사전 정의된 보상 함수 없이 다양한 도메인에서의 정렬

### 도메인 특화 RLVR

범용 RLVR의 한계를 인식하고, 특정 도메인에 최적화된 RLVR 파이프라인이 증가하고 있습니다:
- **의학 RLVR**: 진단 정확성, 치료 안전성을 검증 가능한 보상으로 활용
- **법률 RLVR**: 법령 준수, 판례 일관성을 자동 검증
- **과학 RLVR**: 실험 재현성, 수식 유도의 형식적 검증

## 결론: 연구의 지평선

LLM을 위한 강화학습은 단기간에 놀라운 발전을 이루었지만, 해결해야 할 근본적인 문제들이 남아 있습니다. 이 분야의 발전은 단순히 기술적 도전을 넘어, AI가 인간 사회에 미치는 영향에 대한 깊은 고민을 요구합니다.

가장 중요한 통찰은 다음과 같습니다: **보상 함수는 단순한 목적함수가 아니라 인간의 가치를 코드로 번역한 것입니다.** 그 번역이 얼마나 정확하고 포괄적인지에 따라 AI 시스템의 유용성과 안전성이 결정됩니다. 이 번역을 더 정확하게 만드는 것이 앞으로 수십 년간 AI 연구의 핵심 과제가 될 것입니다.
